This repository demonstrates Kotlin and Spring Boot backend workflows for a Smart ATS project.

- Async resume parsing queue wired through Redis and internal callbacks.
- Recommendation endpoints with match score, semantic score, and XAI reasoning.
- Recruiter review console support for APPLIED, INTERVIEW, and REJECTED states.
- Candidate profile flow enriched with GitHub and portfolio context before parsing.
- PostgreSQL and Docker based local environment for integration testing.
